# YouTube Clone

Clone da interface do YouTube desenvolvido com **HTML**, **CSS** e **JavaScript puro**, reproduzindo o layout da página inicial da plataforma.

## 🔥 Objetivo

Praticar e aprimorar habilidades em:

- HTML semântico
- CSS para layouts responsivos
- Estruturação de interfaces modernas

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do projeto

```
youtube-clone/
└── index.html   # Página principal com toda a estrutura da interface
```

## ✅ Funcionalidades

- Header com barra de pesquisa e ícones de navegação
- Sidebar com menu de categorias
- Grade de vídeos com cards (thumbnail, título, canal, visualizações e data)
- Layout inspirado fielmente na interface do YouTube

## 🧪 Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/wendell0102/youtube-clone.git
   ```
2. Acesse a pasta:
   ```bash
   cd youtube-clone
   ```
3. Abra o arquivo `index.html` no navegador ou utilize a extensão **Live Server** no VS Code.
